# Python Calculator using Tkinter

A simple and interactive calculator application built using **Python** and **Tkinter GUI**. This calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division through a user-friendly graphical interface.

## 📌 Project Overview
This project is a desktop-based calculator designed to simulate the functionality of a standard calculator. It helps users perform quick mathematical calculations with ease while demonstrating GUI development and event-driven programming in Python.

## ✨ Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Decimal number support
- Clear (C) button
- Backspace (⌫) functionality
- Error handling for invalid inputs
- Easy-to-use graphical interface

## 🛠️ Technologies Used
- Python 3
- Tkinter (Python GUI Library)
